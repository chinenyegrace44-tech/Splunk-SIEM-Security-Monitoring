# Splunk-SIEM-Security-Monitoring
Security monitoring project using Splunk SIEM - 5 threat detection use cases built on real log data
### Built with Splunk Enterprise on Kali Linux

### Project Overview
A real -world security monitoring project stimulating a SOC analyst role at Frothly Brewing Company. Analysed 37,362+ security events and detected 5 active threats using Splunk SIEM.

##Environment
- SIEM Platform: Splunk Enterprise 10.2.1
- Operating System: Kali Linux 2025.4
- Lab: VirtualBox VM
-Dataset: Boss dataset

##Threats Detected
#  | Threat                      | Severity               | Status 
1  | Brute Force Attack          | Critical               | Detected
2  | Privilege Escalation        | Critical               | Detected
3  | Data Exfiltration           | High                   | Detected
4  | Malware Detection           | High                   | Detected
5  | Unauthorized Access         | Critical               | Detected

## Key Skilss Demonstrated
- Splunk SPL query writing
- SIEM use case design
- Detection rule configuration
- Alert setup and management
- Security dashboard creation
- Threat investigation and reporting
